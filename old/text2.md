Capítulo 4: El Color - Paletas y Bibliotecas de Color

Un factor muy importante en cualquier ilustración o diseño es el color. Este tutorial le enseñará todo lo que tiene que saber sobre el color en Illustrator. Aprenderá a crear un color personalizado con toda precisión, a crear muestras de color para un documento, a utilizar las bibliotecas y cartas de colores profesionales, a crear sus propias bibliotecas de color, y a extraer colores desde cualquier imagen.

En Illustrator es posible seleccionar colores para las ilustraciones desde una amplia variedad de herramientas, paneles y cuadros de diálogo. Dependiendo de la necesidad de cada momento, se pueden determinar colores mediante el selector de color, panel de muestras, bibliotecas de color y con el cuentagotas.

Selector de color: Mediante esta herramienta se determinan colores personalizados tanto para el relleno como para el trazo de un objeto. Si se hace doble clic en cualquier cuadro del selector de color, se abrirá la ventana del selector de color. En esta ventana, nos aparecen campos como espectro de color, rectángulo de color nuevo, rectángulo de color original, y varios modelos de color. Se puede ajustar el color para que sea más claro u oscuro y más o menos saturado.

Colores compatibles con web: Para ver solo colores compatibles con la web, marque la casilla "Elija solo colores web". Los colores compatibles con la web son los colores utilizados por todos los navegadores web de cualquier plataforma.

Modelos de color: Illustrator maneja varios modelos de color como RGB, HSV, CMYK, entre otros. Cada modelo tiene sus propias características y se utilizan para diferentes propósitos. Por ejemplo, RGB se utiliza para monitores y web, mientras que CMYK se utiliza para impresión.

Bibliotecas de color: Es posible crear y guardar bibliotecas de colores. Esto es útil cuando se tienen colores corporativos o recurrentes en diferentes proyectos. También se pueden añadir colores a las muestras del documento para su uso posterior.

Panel Temas de Color: Este panel permite crear combinaciones de colores armoniosas siguiendo reglas cromáticas como análogos, monocromáticos, triada, complementario, compuesto, tonos y personalizado.
